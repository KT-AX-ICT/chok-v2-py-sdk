# 설계 결정 기록 (ADR)

각 파일은 하나의 결정과 그 배경·대안·결과를 남긴다. 상태: `제안` → `승인` → `대체됨`.

| ADR | 주제 | 상태 |
|---|---|---|
| [ADR-001](ADR-001-snapshot-window.md) | 스냅샷 pre/post 윈도 정의 | 제안 |
| [ADR-002](ADR-002-baseline-source.md) | baseline 프로파일 출처 | 제안 |
| [ADR-003](ADR-003-realtime-signal-scope.md) | 실시간 관측 불가 신호 처리 | 제안 |
| [ADR-004](ADR-004-replayer-data-layout.md) | 리플레이어 데이터 배치 및 원천 경로 | 제안 |

> ADR-002, ADR-003 은 detector 인터페이스를 확정하기 위해 **팀 설계에서 먼저 결론이 필요한 blocker**.
